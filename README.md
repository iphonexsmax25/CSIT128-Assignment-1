# CSIT128 Assignment 1

This project is a simple static HTML webpage for **CSIT128-A1**
## Project Files

- `HTML_CODE.html` – main webpage
- `CPP.jpg` – image used in the C++ section
- `CS.png` – image used in the C# section
- `python.jpg` – image used in the Python section
- `HTML.png` – image used in the HTML section

## How to Run

1. Extract the zip file.
2. Open the project folder.
3. Double-click `HTML_CODE.html`.
4. The page will open in your web browser.

## Requirements

- Any web browser, such as Chrome, Edge, or Firefox.

## Notes

- Keep all image files in the same folder as `HTML_CODE.html`.
- No installation is needed.
- No programming software is needed to run this project.
- If you want to view the code, right-click `HTML_CODE.html` and open it with any IDE or code editor.
